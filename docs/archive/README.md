# Archived drafts — intermediate development material (NOT the final manuscript)

The Markdown files in this folder are **superseded working drafts** from the
development of this study. They are kept only for provenance and transparency.

**They are not the current manuscript.** The authoritative, current text is:

- `../manuscript_final/` — the nine finalized source sections (Abstract,
  Introduction, Methods, Results, Discussion, Conclusion, References + title,
  figure captions, boilerplate). These are the single source of truth.
- `../../manuscript/manuscript_final.docx` — the typeset manuscript compiled
  from those nine files.

These archived files predate the final revision and, in places, contain
**outdated numbers, superseded framing, and internal editorial notes**. In
particular, several were written before the dependent variable was switched to
the frame-count-robust fixed-landmark F0 excursion, and may still cite the older
raw max−min range figures. Do not cite or rely on them.

| `paper_data_pack.md` | Consolidated numbers/data pack used while drafting |
| `results_landmark_consolidated.md` | Landmark-DV results working notes |
| `paper_outline.md` | Section-by-section outline |
| `dv_change_raw_vs_landmark.md` | Notes on the raw→landmark DV change |
