# Language-difference test (cross-linguistic generality)

Study 1 = CSJ Monologue (JP) + Buckeye (EN). DV = F0_excursion_LM_ST (frame-count-robust). N=367390.

## results
- Japanese: 5-95% eff = 1.76 st, slope = 0.250 st/100ms
- English:  5-95% eff = 1.48 st, slope = 0.385 st/100ms
- difference smooth s(Duration):LanguageO p = 6.2e-06
- language level (intercept) term p = 0.159

## verdict: GENERALISES IN MAGNITUDE — the effect is JND-exceeding in BOTH languages (JP 1.76 vs EN 1.48 st, gap 0.27 st). The curve differs 'significantly' (p=6e-06), but at N=367390 this reflects statistical power, not a perceptually meaningful language difference — itself an instance of the significance-vs- effect-size point. Register (Buckeye vs CSJ-Mono) is also unmatched.
